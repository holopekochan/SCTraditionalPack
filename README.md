# Star Citizen Traditional Chinese Pack

This repository contains localization files for Star Citizen. Version: 4.7

## Installation

1. Close Star Citizen if it is running.
2. Locate your Star Citizen `LIVE` directory.
   - Example: `C:\Program Files\Roberts Space Industries\StarCitizen\LIVE`
3. Copy the `data` folder from this repository into the `LIVE` directory.
4. Allow file merge/overwrite when prompted.
5. Launch the game.

## Important

- Back up your original `LIVE\data` files before replacing anything.
- Game updates may overwrite these files, so you may need to copy the pack again after updates.

## Uninstall

- Restore your backup of the original files, or
- Verify/repair the game files using the launcher.

## Repository Structure

- `data/` - Localization and game configuration files
